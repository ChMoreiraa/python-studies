# 🐍 Python Studies

Repositório dedicado à revisão, prática e aprofundamento dos meus conhecimentos em Python.

Aqui serão armazenados exercícios, exemplos, anotações e projetos desenvolvidos ao longo da minha jornada de aprendizado, com foco na aplicação de boas práticas de programação e no desenvolvimento contínuo das minhas habilidades.

## 🎯 Objetivo

Consolidar conceitos da linguagem Python por meio de estudos teóricos e atividades práticas, explorando desde fundamentos até tópicos mais avançados e aplicações em projetos reais.

## 🛠 Tecnologias

- Python 3
- Git
- GitHub
